A simple plug-in that provides a widget to display pretty little (30x30 px) icons with links to your Facebook, Twitter, Google+ and RSS (Feedburner) pages.

== Description ==

This plugin will help you to display links to your Facebook, Twitter, Google+ and RSS (Feedburner) feeds with nice little icons.

These days it's essentially a necessity to operate hub pages on the major social networks, both as a way to improve a sites SEO and to communicate with users and customers across a range of mediums. This plugin will help you to display pretty links to your WordPress Blog or Sites Twitter, Facebook, Google+ and Feedburner (RSS) pages.

At the time of writing we appreciate this plugin is in its infancy and has a very specific usefulness. Stick with us however as we plan to add many more configurable options as part of future versions and we'd love to hear your thoughts via the contact details below. 

== Installation ==

Installation Steps

1. Upload the Social Connect Widget folder to the /wp-content/plugins/ directory via FTP. <strong>OR</strong> Add the Social Connect Widget Plugin via the WordPress Plugins menu.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Widget to the desired footer in the WordPress Widgets menu.
4. Insert your Twitter username, Facebook URL tag, Google+ full url and Feedburner ID and hit save
5. Rate the plugin and verify that it works at wordpress.org.
6. Leave a comment with Scrybes regarding bugs, feature request etc...

Some Formatting Tips

* If you know CSS then you can change the spacing between icons according to your WordPress Blog/Site. Go to WP Dashboard>Plugins>Editor> select Social Connections Widget > Edit social-connect-widget/css/style.css
